== Main Window Layout

Main window uses a `QGridLayout` (`ui/app/mainwindow.ui`) with 2 rows x 2 columns.

=== Grid Cells

- `(0,0)`: application title (`label`)
- `(0,1)`: top custom area (`pageCustomWidgetHost`)
- `(1,0)`: page navigation buttons (`verticalLayout` + `TabButton`)
- `(1,1)`: page content (`stackedWidget`)

Stretch policy is applied in code (`src/app/main_window.cpp`):

- column `1` is stretchable (main content area)
- row `1` is stretchable (main content area)

=== Navigation

`MainWindow` keeps page mapping by enum:

- `Pages::eventInfo`
- `Pages::clientInfo`
- `Pages::clientCard`

Navigation buttons:

- calendar
- clients
- details

Internal helper `showPage(...)` synchronizes:

1. selected page in `QStackedWidget`,
2. checked state for navigation buttons,
3. top custom widget in `(0,1)`.

=== Top Custom Widget API

`MainWindow` exposes:

[source,cpp]
----
void setPageCustomWidget(Pages page, QWidget *widget);
----

Behavior:

- each page can have its own optional widget for cell `(0,1)`,
- `nullptr` means the area is intentionally empty,
- on page switch, `MainWindow` hides old widget and attaches/shows new one.

=== Current Pages

- `QEventInfoPage`: event timeline and event editor dialog.
- `ClientInfo`: list of clients with custom delegate.
- `QClientInfoCardPage`: detailed client card and edit flow.
