== Build And Run

=== Prerequisites

- CMake `>= 3.28`
- Ninja
- C++20 compiler
- Qt6 (Widgets/Core/Gui)
- vcpkg dependencies from `vcpkg.json`

=== Configure

Using preset from `CMakePresets.json`:

[source,bash]
----
cmake --preset vcpkg
----

Manual configure example:

[source,bash]
----
cmake -S . -B build -G Ninja \
  -DCMAKE_TOOLCHAIN_FILE="$VCPKG_ROOT/scripts/buildsystems/vcpkg.cmake"
----

=== Build

[source,bash]
----
cmake --build build
----

=== Run

[source,bash]
----
./build/PsyClientManager
----

=== Tests

Build includes test targets from `test/CMakeLists.txt`.

Run all discovered tests:

[source,bash]
----
ctest --test-dir build --output-on-failure
----

=== Output Paths

- Binary: `build/PsyClientManager`
- UI generated headers: under `build/src/**/.qt/.../ui/...`
