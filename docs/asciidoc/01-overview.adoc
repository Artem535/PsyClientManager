== Overview

`PsyClientManager` is a Qt Widgets desktop application for managing:

- clients,
- events,
- links between clients and events.

Core stack:

- C++20,
- Qt6 (`Core`, `Widgets`, `Gui`),
- DuckDB (embedded DB),
- Poco (filesystem/date utilities),
- reflect-cpp (`rfl`) for config serialization,
- plog for logging.

Entry point:

- `src/main.cpp` creates `pcm::Application` and calls `run(argc, argv)`.

Main runtime flow:

1. `Application` constructs `Database`.
2. `MainWindow` is created.
3. Pages are attached to `MainWindow` (`event`, `clients`, `client card`).
4. Signals connect UI, page logic, and DB operations.
5. Qt event loop starts.
